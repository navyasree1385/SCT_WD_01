# 🚀 Responsive Landing Page

## 📌 Overview

This project is a modern and responsive landing page developed as part of the **SkillCraft Technology Web Development Internship**. The website is designed using HTML, CSS, and JavaScript, focusing on responsiveness, clean UI design, and smooth user interaction.

---

## 🎯 Objective

The objective of this project is to create a visually appealing and fully responsive landing page that provides a seamless user experience across different devices and screen sizes.

---

## ✨ Features

* Fixed Navigation Bar
* Smooth Scrolling Navigation
* Interactive Hero Section
* About Section
* Services Section with Cards
* Contact Section
* Responsive Design for Mobile, Tablet, and Desktop
* Navbar Scroll Effect using JavaScript
* Modern UI with Gradient Backgrounds

## 📖 Project Description

The landing page consists of multiple sections including Home, About, Services, and Contact. A fixed navigation bar allows users to navigate smoothly between sections. The hero section introduces the website with a call-to-action button. The services section highlights key offerings using responsive cards, while the contact section provides communication details.

The website uses CSS media queries to ensure responsiveness on different screen sizes. JavaScript is used to add a dynamic scroll effect to the navigation bar, enhancing the overall user experience.
